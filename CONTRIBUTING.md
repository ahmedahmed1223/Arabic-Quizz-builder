# 🤝 Contributing

Thank you for your interest in contributing to this project!

We welcome all kinds of contributions, including bug fixes, feature requests, documentation improvements, and more.

---

## 🚀 Getting Started

1. Fork the repository
2. Clone your fork:
   ```bashgit clone https://github.com/ahmedahmed1223/Arabic-Quizz-builder.git
cd Arabic-Quizz-builder

3. Create a new branch:

git checkout -b feature/your-feature-name




---

🛠️ How You Can Contribute

Fix bugs

Add new features

Improve documentation

Enhance UI/UX

Optimize performance



---

📋 Guidelines

Write clean, readable code

Use clear naming conventions

Keep changes focused and minimal

Follow the existing project structure and style



---

🧪 Testing

Test your changes before submitting

Make sure nothing else is broken

Add tests if applicable



---

📦 Commit Messages

Use clear and descriptive commit messages:

feat: add new feature
fix: resolve bug
docs: update documentation
refactor: improve code structure


---

📤 Pull Requests

1. Push your branch:

git push origin feature/your-feature-name


2. Open a Pull Request


3. Clearly describe your changes


4. Link related issues if any




---

💬 Communication

Be respectful and constructive

Open an issue first for major changes



---

⚠️ Notes

Do not include sensitive information

Ensure your changes do not break existing functionality

Keep your branch up to date with the main branch



---

🙌 Thank You

Your contributions are greatly appreciated!
