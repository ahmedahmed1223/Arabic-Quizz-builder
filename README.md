# Arabic-Quizz-builder
برنامج لإنشاء الأنواع المختلفة من المسابقات التنافسية الثفافية
المميزات:
لوحة التحكم (كلمة المرور الافتراضية: 1234)

⚙️ الإعدادات — اسم المسابقة، رسالة ترحيب، وقت افتراضي، نظام التنقل، المؤثرات الصوتية
📂 الأقسام والأسئلة — إضافة/تعديل/حذف أقسام بألوان وأيقونات، وأسئلة اختيار من متعدد بوقت مستقل لكل سؤال
👥 الفرق — إدارة الفرق بأسمائها وأعضائها وألوانها ونقاطها
📥 الاستيراد — رفع Excel (8 أعمدة) أو JSON، وتصدير البيانات

وضع العرض

🎬 شريحة افتتاحية مع حلقات متحركة وتأثيرات ضوئية
👥 عرض الفرق ببطاقات ملوّنة
📋 قائمة الأقسام مع مؤشر الأسئلة المتبقية
❓ عرض السؤال مع عداد تنازلي دائري، خيارات، وتمييز بصري للإجابة
🏆 لوحة النتائج مع أشرطة تقدم وميداليات

مزايا تقنية

💾 حفظ تلقائي في localStorage
⌨️ اختصارات لوحة المفاتيح (مسافة/أرقام/أحرف عربية)
🔊 مؤثرات صوتية بـ Web Audio API
🎊 ما تم إضافته
🎮 نمطا المسابقة (في تبويب الإعدادات)
نمط التناوب في الاختيار — الوضع الافتراضي
كل فريق يختار قسمًا → يُعرض سؤال واحد → يُسجَّل النقطة تلقائيًا إن أجاب صحيحًا → ينتقل الدور للفريق التالي ليختار قسمه.
نمط القسم الكامل
الفريق يختار قسمًا → تُوزَّع جميع أسئلة القسم بالتساوي دوريًا على الفرق (السؤال ١ للفريق أ، السؤال ٢ للفريق ب...) → شريط تقدم يوضح الوضع → بعد انتهاء القسم ينتقل الدور للفريق التالي ليختار.
🌟 شريحة الشكر والتقدير

تبويب جديد في لوحة التحكم لإضافة أسماء مقسّمة إلى خمس فئات: إعداد المسابقة — إعداد الأسئلة — الرعاة — الفريق التقني — شكر خاص
لكل شخص: اسم، صفة، لون بطاقة مخصص
شريحة عرض بتصميم احتفالي مع رسالة ختام قابلة للتعديل
زر مخصص في الشاشة الرئيسية والنتائج للوصول إليها مباشرة

📱 تحسين العرض على الشاشات
