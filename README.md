منصة المسابقات التفاعلية (Quiz Platform)

منصة احترافية متكاملة لإعداد وتقديم مسابقات تفاعلية تعليمية وترفيهية. صُممت لتعمل بالكامل داخل المتصفح مع تجربة بصرية غنية ودعم كامل للعمل دون اتصال بالإنترنت أثناء العرض. مثالية للأنشطة العائلية، والنشاطات المدرسية، والفعاليات المؤسسية، والمسابقات الترفيهية.

✨ المميزات الرئيسية

🎮 تجربة عرض احترافية

· واجهة مستخدم مبهرة: دعم كامل لملء الشاشة مع 11 ثيماً بصرياً جاهزاً + ثيم مخصص قابل للتخصيص بالكامل (ألوان الخلفية، اللون الرئيسي، الثانوي، والنص) لمطابقة هوية مؤسستك.
· أنماط مسابقة متعددة: "التناوب في الاختيار" للمنافسات الحادة، و"القسم الكامل" للتعلم التعاوني.
· 5 طرق لعرض الأقسام: شبكة البطاقات، عجلة الحظ الدوّارة، بطاقات مخفية (مفاجآت)، عرض قائمة تفصيلي، ولوحة جيوباردي شبكية — يمكن التبديل بينها في أي وقت.
· أطواق نجاة متكاملة: 50/50، تمديد الوقت (+15 ثانية)، وتمرير السؤال بلا خسارة.
· وضع المُسرِّع (Buzzer): شاشة تنافسية بأزرار كبيرة لكل فريق مع تأثيرات بصرية وجولات متعددة.
· مكافأة التسلسل (Streak Bonus): نقاط إضافية تلقائية عند تحقيق سلسلة إجابات صحيحة متتالية.
· خصم عند الإجابة الخاطئة: نظام خصم نقاط اختياري لزيادة التحدي.

❓ إدارة أسئلة متقدمة

· 8 أنواع من الأسئلة: نصية، صورية، صوتية، معادلات رياضية (KaTeX)، صح/خطأ، أكمل الفراغ، ترتيب، والتوصيل (Match) الجديد كلياً.
· صور على خيارات الإجابة: إرفاق صورة بأي خيار (أ–د) مع إمكانية الجمع بين نص وصورة.
· تخصيص كامل: صعوبة (سهل/متوسط/صعب)، وقت لكل سؤال، وشرح يظهر بعد الكشف.
· خلط وعشوائية: خلط ترتيب الأسئلة، خلط خيارات الإجابة، وبنك الأسئلة (اختيار عدد عشوائي لكل جلسة).
· بحث وتصفية: شريط بحث نصي + فلاتر حسب الصعوبة والنوع + نسخ الأسئلة بضغطة واحدة.
· استيراد سريع: سحب وإفلات ملفات Excel (xlsx/xls/csv).

🖥️ شاشات العرض

· عداد دائري متحرك: قابل للإيقاف مع مؤشر بصري للوقت المتبقي، وتأخير اختياري لبدء العدّاد.
· كشف تدريجي للخيارات: إظهار الخيارات واحداً تلو الآخر يدوياً أو تلقائياً بمدة زمنية قابلة للضبط.
· تأكيد الإجابة قبل القفل: لتجنب الأخطاء غير المقصودة.
· أزرار تحكّم للمضيف: تقييم مباشر (صح/خطأ) في أسئلة أكمل الفراغ والترتيب.
· منصة التتويج (Podium): شاشة نهاية احتفالية بمراكز ذهب–فضة–برونز، كونفيتي، وموسيقى مخصصة.
· شاشة الشكر والتقدير: ثلاثة أنماط عرض — عادي، سينمائي (نهايات الأفلام)، وبطاقات تفاعلية قابلة للقلب.
· الساعة الكبيرة (Big Clock): عداد مكبّر للقاعات الكبيرة.
· الإحصاءات التفصيلية: أداء كل فريق بالتفصيل — إجابات صحيحة/خاطئة، متوسط الوقت، ونسبة الدقة.

🔊 نظام صوتي وتفاعلي

· موسيقى تفاعلية: 3 أنماط مدمجة تتغير تلقائياً حسب حالة الوقت (هادئة، نشيطة، توتر).
· موسيقى مخصصة: رفع ملفات خارجية للموسيقى الخلفية، أصوات الإجابات، موسيقى العجلة، وموسيقى المنصة.
· منسّق صوتي مركزي: يضمن عدم تداخل الأصوات مع خاصية خفض الموسيقى مؤقتاً (ducking).
· شريط موسيقى عائم: في جميع شاشات العرض للتحكم السريع.

👥 إدارة الفرق

· تخصيص متكامل: اسم، لون، شعار (صورة)، وأعضاء مع صور شخصية.
· تعديل النقاط يدوياً: أزرار +1 / −1 / صفر مع خاصية التراجع عن آخر تغيير وسجل تدقيق.

🛠️ أدوات التحكم والإدارة

· لوحة تحكم محمية بكلمة مرور قابلة للتغيير.
· صور الأقسام والفرق: رفع صور مخصصة.
· اختصارات لوحة المفاتيح: Space/Enter للكشف والانتقال، 1–4 لاختيار الإجابة، Esc للإلغاء، وحماية من F5.
· إعادة ترتيب بالسحب: سحب وإفلات لترتيب الأقسام، الفرق، الأسئلة، والخيارات — يعمل باللمس على كل الأجهزة.
· إشعارات المتصفح: تنبيهات سطح مكتب للمقدم عند انتهاء الوقت حتى لو كان المتصفح في الخلفية.

💾 التقنية والأمان

· حفظ تلقائي (Offline First): كافة البيانات تُخزّن محلياً مع استمرارية بعد الإغلاق.
· نسخ احتياطي تلقائي: تنزيل بفترات قابلة للتخصيص مع إمكانية نسخ فوري.
· تصدير/استيراد JSON وتصدير مشفّر: نقل المسابقة كاملة بين الأجهزة بأمان.
· واجهة برمجية (API): كائن QuizV7 يسمح للمستخدمين المتقدّمين بالتحكم بالنقاط، الصوت، المؤقتات، والأحداث من وحدة التحكم.
· حماية العرض: حجب تلقائي لأزرار التحديث أثناء المسابقة وتأكيد الخروج.
· متوافق مع كل الشاشات: هواتف، أجهزة لوحية، حواسيب، شاشات عرض 4K.
