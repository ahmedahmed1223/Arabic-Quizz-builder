# 🚀 منصة المسابقات التفاعلية (Quiz Platform)

منصة احترافية متكاملة لإعداد وتقديم مسابقات تفاعلية تعليمية وترفيهية.  
صُممت لتعمل بالكامل داخل المتصفح مع تجربة بصرية غنية ودعم كامل للعمل دون اتصال بالإنترنت أثناء العرض.  
مثالية للأنشطة العائلية، والنشاطات المدرسية، والفعاليات المؤسسية، والمسابقات الترفيهية.

---

## ✨ المميزات الرئيسية

### 🎮 تجربة عرض احترافية
- واجهة مستخدم مبهرة: دعم كامل لملء الشاشة مع **15 ثيماً بصرياً جاهزاً** + ثيم مخصص قابل للتخصيص بالكامل (ألوان الخلفية، اللون الرئيسي، الثانوي، والنص) لمطابقة هوية مؤسستك، مع دعم حفظ **لوحات ألوان مخصصة متعددة**.
- أنماط مسابقة متعددة:  
  - "التناوب في الاختيار" للمنافسات الحادة  
  - "القسم الكامل" للتعلم التعاوني
- 5 طرق لعرض الأقسام:
  - شبكة البطاقات  
  - عجلة الحظ الدوّارة  
  - بطاقات مخفية (مفاجآت) مع إعادة خلط تلقائي بعد كل سؤال
  - عرض قائمة تفصيلي  
  - لوحة جيوباردي شبكية  
  > يمكن التبديل بينها في أي وقت
- أطواق نجاة متكاملة: 50/50، تمديد الوقت (+15 ثانية)، وتمرير السؤال بلا خسارة
- وضع المُسرِّع (Buzzer): شاشة تنافسية بأزرار كبيرة لكل فريق مع تأثيرات بصرية وجولات متعددة
- مكافأة التسلسل (Streak Bonus): نقاط إضافية تلقائية عند الإجابة الصحيحة المتتالية
- خصم عند الإجابة الخاطئة: نظام اختياري لزيادة التحدي

---

### ❓ إدارة أسئلة متقدمة
- **9 أنواع من الأسئلة:**
  - نصية
  - صورية
  - صوتية
  - معادلات رياضية (KaTeX)
  - صح / خطأ
  - أكمل الفراغ
  - آية قرآنية (مع جلب تلقائي وخيار القارئ)
  - ترتيب
  - التوصيل (Match)
- صور داخل الخيارات (A–D) مع إمكانية دمج نص + صورة
- تخصيص كامل: صعوبة، وقت، وشرح بعد الإجابة
- خلط وعشوائية: الأسئلة + الخيارات + بنك أسئلة
- بحث وتصفية متقدمة أثناء اللعب (نوع + صعوبة)
- استيراد سريع عبر Excel (xlsx / xls / csv)
- **مكتبة أسئلة مدمجة:** 100 سؤال جاهز في 10 أقسام بثلاثة أوضاع تحميل (استبدال / إضافة / دمج ذكي)

---

### 🖥️ شاشات العرض
- عداد دائري متحرك مع مؤشر بصري للوقت
- كشف تدريجي للخيارات (يدوي أو تلقائي)
- تأكيد الإجابة قبل الإغلاق
- تحكم مباشر للمضيف في التصحيح
- **سبورة تفاعلية** كاملة الشاشة للرسم والتوضيح أثناء العرض
- منصة التتويج (Podium):
  - ذهب – فضة – برونز
  - مؤثرات احتفالية + كونفيتي + موسيقى
- **شهادة الفائز** قابلة للطباعة بالتاريخ الهجري والميلادي وشعار مخصص
- شاشة شكر احترافية:
  - عادي
  - سينمائي (نهايات أفلام)
  - بطاقات تفاعلية قابلة للقلب
- ساعة كبيرة للقاعات
- إحصاءات تفصيلية لكل فريق مع مقارنة 1v1 وتصدير CSV

---

### 🔊 نظام صوتي وتفاعلي
- موسيقى ديناميكية حسب الحالة (هادئة / نشطة / توتر)
- **خلاط صوت 6 قنوات مستقلة:** خلفية، مؤثرات، عجلة، سؤال، توتر، منصة
- دعم رفع موسيقى مخصصة بالكامل
- منسّق صوتي مركزي مع تقليل تلقائي للصوت (Ducking)
- شريط تحكم صوتي عائم في كل الشاشات
- موسيقى توتر مستقلة تُفعَّل تلقائياً في آخر N ثانية من العداد

---

### 👥 إدارة الفرق
- تخصيص كامل:
  - اسم الفريق
  - لون
  - شعار
  - أعضاء وصور شخصية
- تعديل النقاط:
  - +1 / −1 / صفر
  - تراجع عن آخر تعديل
  - سجل تغييرات كامل
- نظام مكافأة السلسلة والخصم عند الخطأ

---

### 🛠️ أدوات التحكم والإدارة
- لوحة تحكم محمية بكلمة مرور
- **وضع التحكم عن بُعد** عبر BroadcastChannel API
- رفع صور للأقسام والفرق
- اختصارات لوحة مفاتيح:
  - Space / Enter: انتقال أو كشف
  - 1–4: اختيار الإجابة
  - Esc: إلغاء
- سحب وإفلات لإعادة الترتيب (يدعم اللمس)
- إشعارات سطح المكتب عند انتهاء الوقت
- حماية من التحديث أثناء العرض (F5)
- **أنماط عرض تقديمي:** مباشر / تسلسلي تلقائي / تسلسلي يدوي

---

### 💾 التقنية والأمان
- Offline First: يعمل بدون إنترنت
- **سياسة أمان المحتوى (CSP)** ومُطهِّر HTML مدمج ضد XSS
- **تخزين أساسي في IndexedDB** مع localStorage كاحتياطي مضغوط
- **ضغط LZ-String** للبيانات (أصغر بـ 70%)
- حفظ تلقائي مستمر مع فحص التغييرات
- نسخ احتياطي دوري + يدوي
- تصدير/استيراد JSON
- تصدير مشفّر لنقل آمن
- API برمجي (BroadcastChannel) للتحكم الكامل
- حماية من الخروج غير المقصود
- توافق كامل مع جميع الأجهزة (جوال / تابلت / كمبيوتر / 4K)
- دعم `prefers-reduced-motion` لإمكانية الوصول

---

## 🚀 كيف تبدأ؟
1. قم بتحميل ملف: `quiz-platform.html`
2. افتحه باستخدام Chrome أو Edge
3. سجّل الدخول باستخدام كلمة المرور الافتراضية: **1234**
4. ابدأ بإعداد مسابقتك

> ⚠️ **تنبيه أمان:** غيّر كلمة المرور الافتراضية قبل أي استخدام في بيئة عامة.
