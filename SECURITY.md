# 🔒 Security Policy

## 📦 Supported Versions
The following versions are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| >= 7.0  | ✅                 |
| < 5.0   | ❌                 |

> ⚠️ Versions between **5.0 – 6.x** may have limited or no security support.

---

## 🚨 Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

Since this is a hobby project, security updates and fixes will be addressed **as quickly as possible**, but response times may vary.

### 📬 How to Report
- Contact the project maintainer directly (e.g., via email or GitHub Issues).
- Provide a clear description of the vulnerability.
- Include steps to reproduce the issue (if applicable).
- Attach screenshots or proof-of-concept when possible.

### ⏱️ What to Expect
- You will receive an acknowledgment within a few days.
- Updates on the status of the report will be shared periodically.
- If the vulnerability is accepted, a fix will be developed and released.
- If declined, a clear explanation will be provided.

---

## 🙏 Responsible Disclosure
Please do **not** publicly disclose the vulnerability until it has been reviewed and resolved.

Thank you for helping improve the security of this project.
